# The definitive LandFix for Sourcemod bhop servers
This is the FiH version of LandFix that does not make you lose time while enabled.
<p>Compared with no plugin, it will yield the same time, but with more consistent 64u bhops. Alternatives (like Haze's) will not allow you to gain the normal amount of speed, and must be toggled on/off constantly to get their benefit without losing speed.</p>
<p>This version is set and forget, no need to enable or configure anything. Just upload the plugin and enjoy more consistent jumps without drawbacks! (and without mechanic changes.. just an engine quirk/bug fixed)</p>
